Hugo 主题 Stack 支持使用 Markdown 创建交互式图片画廊。它由 [PhotoSwipe](https://photoswipe.com/) 提供支持，语法灵感来自 [Typlog](https://typlog.com/)。

要使用此功能，图片必须与 Markdown 文件位于同一目录，因为它使用 Hugo 的页面捆绑功能读取图片尺寸。**不支持外部图片。**

## 语法

```markdown
![Image 1](1.jpg) ![Image 2](2.jpg)
```
